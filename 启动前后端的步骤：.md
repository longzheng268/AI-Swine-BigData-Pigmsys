启动后端的步骤

打开idea---文件---设置--搜索maven---找到用户设置文件导入本地的maven的settings.xml和bin文件

等待解析maven依赖项   最后运行项目 无任何报错即为成功



## 启动前端的步骤：

1. **进入前端项目目录**：

   ```cmd
   cd viewPart\pig-msys
   ```

2. **安装依赖**（如果还没有安装）：

   ```cmd
   npm install
   ```

3. **启动开发服务器**：

   ```cmd
   npm run serve
   ```
