# 启动前后端的步骤

## 启动后端的步骤

1. **配置 Maven**：
   - 打开 IDEA → 文件 → 设置 → 搜索 Maven
   - 找到用户设置文件导入本地的 Maven 的 settings.xml
   - 确认 Maven 主目录配置正确

2. **等待解析 Maven 依赖项**：
   - IDEA 会自动下载和解析项目依赖
   - 等待依赖下载完成

3. **配置数据库连接**：
   - 编辑 `pigms/src/main/resources/application.yml`
   - 配置 MySQL 连接信息（数据库地址、用户名、密码）
   - 配置 Hadoop 地址（如果使用大数据分析功能）

4. **运行项目**：
   - 找到主启动类运行
   - 或使用命令：`cd pigms && mvn spring-boot:run`
   - 后端服务运行在 http://localhost:8080
   - 无任何报错即为启动成功

## 启动前端的步骤

1. **进入前端项目目录**：

   ```bash
   cd viewPart/pig-msys
   ```

2. **安装依赖**（首次运行或依赖更新时）：

   ```bash
   npm install
   ```

3. **启动开发服务器**：

   ```bash
   npm run dev
   ```

   前端服务将运行在 http://localhost:8081

4. **默认登录账号**：
   - 用户名: `admin`
   - 密码: `123456789`

## 注意事项

- 确保已安装 JDK 11 或更高版本
- 确保已安装 Node.js 16.x 或更高版本
- 确保 MySQL 8.0 服务已启动并导入了数据库
- 前端通过代理访问后端 API，确保后端服务先启动
- **无需启动 Python 服务**：预测功能已迁移到前端实现
