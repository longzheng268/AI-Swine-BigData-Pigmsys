## 技术栈总览

### 后端（Java）

- 核心框架：Spring Boot 2.6.7

- 开发语言：Java 1.8

- ORM：MyBatis 2.2.2

- 数据库：MySQL（使用 Druid 连接池 1.2.8）

- 安全认证：Spring Security + JWT（jjwt 0.9.1）

- 大数据处理：Hadoop 2.10.2（MapReduce）

- 工具库：

- Lombok

- PageHelper 1.4.2（分页）

- Apache POI 5.2.2（Excel）

- FastJSON 1.2.75（JSON）

- Hutool 5.8.16（工具）

- Log4j 1.2.17

- AOP：Spring AOP（日志）

### 前端（Vue）

- 框架：Vue 3.3.4

- 路由：Vue Router 4.2.5

- 状态管理：Vuex 4.1.0

- UI组件库：Element Plus 2.4.2

- 图表：ECharts 5.3.2

- HTTP客户端：Axios 0.27.2

- 构建工具：Vue CLI 5.0.8

- 样式：Sass

- 开发端口：8081（代理到后端 8080）

### Python 服务

- Web框架：Flask 2.3.0

- 跨域：Flask-CORS 4.0.0

- 机器学习：scikit-learn 1.3.0

- 数值计算：NumPy 1.24.3

- 模型持久化：Joblib 1.3.1

- 服务端口：5000

### 项目架构

- 前后端分离：Vue 3 前端 + Spring Boot 后端

- 微服务：独立的 Python 预测服务

- 大数据处理：集成 Hadoop MapReduce 进行数据分析
